О нас

О проекте

Этот сайт создан в рамках изучения современных технологий развертывания статических сайтов с использованием CDN и автоматизированных процессов CI/CD.

Мы исследуем возможности отечественных CDN-провайдеров и их интеграцию с популярными платформами для хостинга статических сайтов.

Технологии

Фронтенд

  • • Jekyll - генератор статических сайтов
  • • Tailwind CSS - утилитарный CSS-фреймворк
  • • PostCSS - инструмент для обработки CSS
  • • JavaScript - минимальная интерактивность

Инфраструктура

  • • GitHub Actions - CI/CD автоматизация
  • • GitHub Pages - хостинг статических сайтов
  • • CDN - ускорение доставки контента
  • • HTML валидация и минификация

Цели проекта

1

Изучение возможностей отечественных CDN-провайдеров для развертывания статических сайтов

2

Создание автоматизированного пайплайна сборки и деплоя с валидацией HTML

3

Демонстрация современных подходов к разработке статических сайтов

4

Оптимизация производительности и SEO-показателей

CDN и облачные технологии
